# Excel-Data-Analysis-Project
Analyzed Vrinda Stores’ 2022 sales data to understand customer behavior and revenue drivers. Identified higher purchases by women, strong contribution from the 39–40 age group, top states, and major sales via Amazon, Flipkart, and Myntra to support 2023 growth strategy.

## Objective
Analyze train-related data using Excel to derive meaningful insights.

## Tools Used
- Microsoft Excel
- Pivot Tables
- VLOOKUP / IF functions
- Charts & Dashboard

## Project Description
This project focuses on analyzing structured data using Excel. The dataset was cleaned, analyzed, and visualized to identify trends and patterns.

## Key Learnings
- Data cleaning in Excel
- Pivot table analysis
- Dashboard creation
